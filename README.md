# 🐄 QurbaniHat – Livestock Booking Platform

## 🌐 Live URL
https://b13-a08-chi.vercel.app/

## 🎯 Purpose
QurbaniHat is a modern web application where users can explore livestock (cows, goats, etc.) for Qurbani, view detailed information, and place bookings after authentication.

## ✨ Key Features
- Browse and explore animals with details  
- Sort animals by price  
- Booking system (login required)  
- User authentication (Email/Password + Google)  
- Profile page with update option  
- Toast notifications & loading states  
- Fully responsive design  
- Custom 404 Not Found page  

## 📦 NPM Packages Used
- react-router-dom  
- firebase  
- axios  
- react-hot-toast / react-toastify  
- tailwindcss  
- daisyui  
- animate.css / lottie-react / react-spring
